## 4. Must Haves

- A user should be able to enter a city name and see city's name, country, degree and max-min degree.

- A user should be able to see date as analog.

- A user should be able to see no data notification if the city name is invalid.

## 5. Nice-to Have

- After a user enter a city name search bar will be cleaned.

- A user always see a video background running autoplay and infinite.

- The app has a fav-icon.
